# 🕹️ Petris

**Version:** `v1.0`  
**Date:** _(Date not specified)_  

### 🎯 Purpose
**Petris** is a custom arcade-style game inspired by the classic mechanics of falling block games. Designed as a Java-based desktop application, Petris offers a fun, interactive experience where players strategically place blocks to maximize space and score.

---

### 🚀 Getting Started

1. Navigate to the folder structure:  
   `Petris -> src -> main`
2. Open `Main.java` in your preferred Java IDE (e.g., [Visual Studio Code](https://code.visualstudio.com/), IntelliJ, Eclipse).
3. Compile and run the `Main.java` file to launch the game.

---

### 🕹️ How to Play

- Gameplay instructions are integrated into the game screen.
- Use your keyboard to control block movements and positioning.
- Try to **optimize your space** and **avoid stacking too high**.
- The game ends if the pieces overflow the top of the board.
